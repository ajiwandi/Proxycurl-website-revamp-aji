Company API

Enrich LinkedIn company profiles; get funding data, list and search employees, identify companies, get employee count.

Read API Docs for Company API

Company Lookup Endpoint

Identify a company by a name or domain

Input
http://apple.com 
                                    
  curl \ -G \ -H "Authorization: Bearer ${YOUR_API_KEY}" \ 'https://nubela.co/proxycurl/api/linkedin/company/resolve' \ --data-urlencode 'company_domain=http://apple.com' \

Output
LinkedIn URL
linkedin.com/company/apple 
                        
  {
    "url": "https://sg.linkedin.com/company/apple"
  }

Company Profile Endpoint

Enrich a company profile with structured data

Input
linkedin.com/company/apple 
                                  
  curl \ -G \ -H "Authorization: Bearer ${YOUR_API_KEY}" \ 'https://nubela.co/proxycurl/api/linkedin/company' \ --data-urlencode 'url=https://www.linkedin.com/company/apple/' \ --data-urlencode 'resolve_numeric_id=true' \ --data-urlencode 'categories=include' \ --data-urlencode 'funding_data=include' \ --data-urlencode 'extra=include' \ --data-urlencode 'exit_data=include' \ --data-urlencode 'acquisitions=include' \ --data-urlencode 'use_cache=if-present'
